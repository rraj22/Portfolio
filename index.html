Deep Learning Applications

Key Point of Project

To analyze various Convolutional Neural Network (CNN) architectures for classifying laminar and turbulent flow in simulation images. A comprehensive dataset of 2D pipe flow images was developed using COMSOL simulations. Several CNN architectures, including ResNet50, VGG16, AlexNet, LeNet5, and MobileNet, were implemented and evaluated. The project successfully resulted in a model capable of classifying laminar and turbulent flow images with approximately 86% accuracy, providing valuable insights into the behavior of fluid flows through deep learning techniques.
